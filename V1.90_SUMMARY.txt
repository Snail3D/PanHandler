═══════════════════════════════════════════════════════════════════════
                         HOW IT WORKS
═══════════════════════════════════════════════════════════════════════

1. USER LAUNCHES APP
   → Random color set selected (e.g., Blue/Amber/Pink)
   → Camera screen shows:
     - Blue crosshairs
     - Amber bubble level
     - Pink shutter button

2. USER TAKES PHOTO
   → Transitions to calibration screen
   → Circle/ring uses BLUE (matches camera crosshair)
   → Visual continuity maintained

3. USER COMPLETES CALIBRATION
   → Enters measurement screen
   → Session colors persist (no regeneration)

4. USER TAPS "NEW PHOTO"
   → Returns to camera mode
   → NEW random color set selected (e.g., Purple/Green/Orange)
   → Fresh visual experience

5. USER TAPS "RECALIBRATE"
   → Returns to calibration screen
   → SAME colors (stays in session)
   → No regeneration
