📐 WHAT IS PANHANDLER?

Professional measurement tool for iOS that uses coin-based 
calibration to deliver CAD-ready accuracy. No hardware needed.

═══════════════════════════════════════════════════════════════

✨ CORE FEATURES

Measurements:
  ✅ Distance (linear, two points)
  ✅ Angle (three-point, vertex-based)
  ✅ Circle (radius, diameter, area, circumference)
  ✅ Rectangle (dimensions, area, perimeter)
  ✅ Freehand (irregular shapes, area/perimeter) - PRO

Calibration:
  ✅ Coin calibration (50+ international coins)
  ✅ Map scale (blueprints, drawings, maps)
  ✅ Remembers last coin used

Export:
  ✅ Email (annotated image + data)
  ✅ Save to Photos (measurements burned in)
  ✅ CAD export (DXF format) - PRO
  ✅ Custom labels

UX:
  ✅ Watery glassmorphic design
  ✅ Dynamic color system (7 vibrant colors)
  ✅ Gesture controls (pan, pinch, swipe, shake)
  ✅ Haptic feedback throughout
  ✅ Comprehensive help system
  ✅ Offline & lightweight

🧮 CORE MATHEMATICS

Calibration:
  pixelsPerMM = (referenceRadius * 2 / zoomScale) / coinDiameterMM

Distance:
  realWorldDistance = √[(x2-x1)² + (y2-y1)²] / pixelsPerUnit

Angle:
  angleDegrees = acos(dotProduct / (mag1 * mag2)) * 180/π

Circle:
  area = π * radius²
  circumference = 2π * radius

Rectangle:
  area = width * height
  perimeter = 2(width + height)

Freehand:
  area = Shoelace algorithm
  perimeter = sum of edge distances
